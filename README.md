# 💧 Water Quality Prediction System

This project uses Machine Learning to predict whether water is safe for drinking based on various chemical parameters.

## 📌 Features
- Data preprocessing and cleaning
- Machine learning model training
- Water quality prediction
- Simple Python-based interface

## 🧠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## 📂 Project Files
- train_model.py → Model training
- water_quality_prediction.py → Prediction script
- water_potability.csv → Dataset
- water_quality_model.pkl → Trained model

## 🚀 How to Run
```bash
pip install -r requirements.txt
python water_quality_prediction.py
